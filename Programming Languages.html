    Term Definition
    debugging a branch of artificial intelligence (AI) and computer science which focuses on the use of data and algorithms to imitate the way that humans learn, gradually improving its accuracy.
    assembler a type of computer program that takes in basic instructions and converts them into a pattern of bits that the computer's processor can use to perform basic operations.
    assembly language s a low-level computer language where the commands are more close to machine level language and equally understandable to human also.
    translator returns a string where some specified characters are replaced with the character described in a dictionary, or in a mapping table..
    compiler system software (a set of a computer program) that converts source code written in a programming language (source language, usually HLL) into another computer language (target language). .
    interpreter system software which has the capability to read and execute the program,
    cross-platform software an approach to developing software applications that are compatible with multiple mobile operating systems (OSes) or platforms
    natural language a machine learning technology that gives computers the ability to interpret, manipulate, and comprehend human language.
    formal language is a set of strings of symbols drawn from a finite alphabet. .
    machine independence A term applied to software that is not dependent on the properties of a particular machine, and can therefore be used on any machine.
    high-level programming language has a significant abstraction from the details of computer operation.
    object-oriented programming creating objects that contain both data and functions.
    scripting language an advanced scripting language that is being used successfully to glue together large software components
    integer a whole number, positive or negative, without decimals, of unlimited length.
    float a function or reusable code in the Python programming language that converts values into floating point numbers.
    string sequences of character data.
    primitive ntegers, floats, booleans and strings.
    data structure mutable (lists, dictionaries, and sets) and immutable (tuples)
    python collection data structures List is a collection which is ordered and changeable.
    list Lists are used to store multiple items in a single variable.
    tuple used to store multiple items in a single variable.
    set Set is one of 4 built-in data types in Python used to store collections of data,
    Protection the process of protecting sensitive information from damage, loss, or corruption
    dictionary A dictionary in Python is a collection of key-value pairs
    function The Spotfire Service for Python provides execution environments for running Python scripts in a Python interpreter installed on the node manager.

    global variable Variables that are created outside of a function
