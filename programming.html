    phases of computer program devlopment: Phase 1: understand the proble Phase 2: Design an algorithm to solve the problem Phase 3: Write the program code in appropriate language based on algorithm Phase 4: Enter the program and save Phase 5: Execute the program Phase 6: Test the program, find errors and debug, then retest Phase 7: Document and evaluate the program
    Programming structure: Sequential structure: top-down execution Iterative (Looping) While loop: continue looping as long something is true Repeat/until loop: continue looping as long something is true For loop: loop a given number of times
    Recursive structures: Repeat a set of instructions as a subtask of itself
    Repetition continues until: Degenerative case, which is termination codition, is met (Degenerative case is also called a "base case")
    Two classic programming/database problems : Sorting: Descending order/Ascending order Sorting numbers Storting strings (in accordance with ASCII values) Searching: find specific data that meet a certain criteria
    In databse: Filters: temporary search Queries: search criteria are saved so they dont have to be recreated
    Two popilar search techniques: Sequential search: search from top to bttom or bottom to top, one item at a time
    Two classic program goals: Efficiency: program takes minimal time and/or resources Correctness: program runs expected ways and works for all expected coditions
