    Term Definition
    Operating system The software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
    Job queue A list of tasks to be carried out in turn by an operating system.
    FIFO Is an accounting method in which assets purchased or acquired first are disposed of first. FIFO assumes that the remaining inventory consists of items purchased last.
    Interactive processing A computer system, the process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information.
    Time-sharing multiprogramming The sharing of computing resources among several users simultaneously.
    Application software Software designed to handle specific tasks for users.
    System software A type of computer program that is designed to run a computer's hardware and application programs.
    Utility software Utility software is a Antivirus, backup software, file manager, disk compression tool
    Graphical user interface (GUI) A type of user interface through which users interact with electronic devices via visual indicator representations.
    Shell UNIX term for the interactive user interface with an operating system.
    Kernel A core component of an operating system and serves as the main interface between the computer's physical hardware and the processes running on it.
    Device driver A set of files that tells a piece of hardware how to function by communicating with a computer's operating system.
    Virtual memory A common technique used in a computer's operating system (OS).
    Bootstrapping Rom Read-only memory.
    Deadlock A situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function.
    Spooling A process in which data is temporarily held to be used and executed by a device, program or the system.
    Super user (Or Administrator) Highly privileged accounts primarily used for administration by specialized IT employees.
    Multi-Factor Authentication A multi-step account login process that requires users to enter more information than just a password.
